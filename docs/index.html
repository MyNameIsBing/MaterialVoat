MaterialVoat

MaterialVoat is a css theme for Voat which sort of follows Google's Material design guidelines. This is still in development so there will be bugs. Please report any bugs you find here.

For subverse moderators

You may use this in any subverse.

Download

Installing on your subverse

If you want to change the color of the CSS, the best way to do so is by using Ctrl + f and replacing them with the colors you want. Keep in mind, though you can use any color, it is best if you use ones from the material design color palette . By default, the theme is set to
500 Blue #2196f3
as the body color,
800 Blue #1565C0
as the status bar color, with
A700 Amber #ffab00
and
A700 Amber #ffc400 (for hover)
as the accent color. For dark mode, the accent colors are
A700 Red #d50000
and
A400 Red #ff1744

Addons

I have created a few addons for you. Feel free to submit your own to the subverse.

  • Header Image

  • Just add this after the full css. You can then replace "http://imgur.com/FwxGrgy.png" with anything.

  • Sticky Link

  • Just add this after the full css. To use, add "#####[ANNOUNCEMENTTEXT](ANNOUNCEMENTLINK)" to your sidebar.

  • Side Link Button

  • Just add this after the full css. To use, add "####[BUTTONTEXT](BUTTONLINK)" to your sidebar.
